How does JavaScript Engine work?

JavaScript create pana apo adhu interpreted language ah tha irunduchi because javascript engine interpreter use pani code execute panuchi but, now-a-days all modern browsers oru javascript code execute pana interpreter & JIT(just-in-time) compiler use pani execute panudhu. Step by step ah pakulam epdi js code execute agudhu nu

  • Parsing: JavaScript engine nambo eluzhudura script la padichi oru tree matri form panum adhuku peru AST(Abstract Syntax Tree) nu soluvanga idhuku peru tha parsing.
  • Interpretation: Then interpreter ena panum na anda AST ya byte code ah convert panum.
  • JIT: JIT compiler ena panum na evolo ku evolo code optimize pana mudiumo avolo ku avolo optimize panum.
  • Execution: Finally, JIT optimized pana code execute panum.

    • Note: Different JS engine has different architectures. For example, Chrome has V8 engine adhu vechithu irukura compiler name TurboFan, Mozilla's has SpiderMonkey engine nadhu vechithu irukura compiler name JIT compiler.
