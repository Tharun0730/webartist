How to insert JavaScript into an HTML page

There are two ways to insert JavaScript program in HTML page.

  • Internal JavaScript
  • External JavaScript

Internal JavaScript

Internal JavaScript use panum podu nambo HTML la direct ah <script> tag kula js code la podhuvom.

<!DCOTYPE html>

<html lang = "en">

<head>

<title>Internal JavaScript<//title>

</head>

<body>

<!--HTML code here-->

<script>

//JavaScript code here

</script>

</body>

</html>

Nambo <script> tag HTML la enga vena podhalam but mostly <body> tag kula or <head> tag kula podhuvanga rendhuthukum neriya difference iruku.

Placing script in head tag

<!DCOTYPE html>

<html lang = "en">

<head>

<title>Script in Head</title>

<script src = "script.js" ></script>

</head>

<body>

<h1>webArtist</h1>

</body>

</html>

<script> tag head kula use panum podhu code execute agum podu first <script> tag read pani execute pana piragu tha html content display agum. Imagine that js code rombo adigam ah irukum podhu website loading time rombo adigam agum, html code display agurathukum late agum so, users experience affect panum.

It is not good practice to place <script> tag inside head tag.

Placing script at the end of the body tag

Previous method la irukura issue solve pana tha mostly ella developers <script> tag body kula use panuvanga.

Next, <body> tag kula use pananum nu solitha but <body> tag kula enga podrathu nu doubt vanda, as i said HTML content display ahna piragu <script> tag execute achi na loading time konjam kami agum at the same time user experience affect agadhu.

So, i recommend to use <script> tag inside <body> tag before body tag ends like this ðŸ‘‡

<!DCOTYPE html>

<html lang = "en">

<head>

<title>Script in Head</title>

</head>

<body>

<h1>webArtist</h1>

<script src = "script.js" ></script>

</body>

</html>

3 Ways to print webArtist in JavaScript

Using console.log ()

console.log() is a function used to log a message in console.

Program la function na oru input vangi process pani output kudukuradhu tha function nu soluvanga.

For example, real world la ipo marriage function edhuthukithom na anda function epdi la irukanum nu nambo decide panuvom and adhuku ahna velai la panuvom.

Nambo oruthar aala marriage function pana mudiyathu so adhuku nu oru wedding planner choose panuvom avanga kita nambo inda matri la function nadakanum nu soluvom adhu tha input adhu process pandrathavanga tha wedding planner avnaga tha inda scenario la function nambo sona matri decorate pani kudurathu tha output.

Ipo nambo console.log use pani webArtist print panalam.

<script>

console.log("webArtist");

</script>

To see this output unga HTML file browser la open panunga and mukiyam ah mela irukura script html file include pani irukanum.

Browser la open pana piragu press F12 or right-click and select inspect developer tools open agum anga elements tab pakathula console tab click panunga, here you can see your output.

document.write() idhuvum oru function. But, console.log use pana developer tools la log pani tha paak mudium, idhuvae document.write function use pana HTML document la write panalam.

<script>

Idhu odha output paaka HTML file browser open panunga, inda script unga HTML file la write panithu.

Using alert()

alert() same idhuvum oru function, but inda function ena panum na HTML file load aguna odhane oru pop up box open agum adhula nambo alert la ena message kudukuromo anda message display agum.

JavaScript Comments

JavaScript la comments edhuku use agudhu na, nambo eluzhudura code innoru developer edhuthu pathalum avangaluku puriyanum nu and nambo eluzhudhura code nambalukae sila time puriyadhu oru 1 or 2 day kalichi edhuthu partha so adhuku nambaluku puriyura matri eluzhudi vechikulam like notes matri.

Nambo eluzhudura comment ku munnadi rendu // forward slash potha adhu tha single line comment. Oru line mathum comment pananum nu nenaicha idhu use panikulam.
